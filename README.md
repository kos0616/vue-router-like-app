# vue-router-like-app

Let vue-router transfer like an APP!
[demo](https://kos0616.github.io/vue-router-like-app/)

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
